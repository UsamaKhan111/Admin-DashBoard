# Admin Dashboard UI

A modern and fully responsive admin dashboard built using **HTML**, **CSS**, and **JavaScript**.  
This project features a clean UI, responsive layout for all screen sizes, animated sidebar navigation, and a fully functional dark/light theme toggle.

-  Fully Responsive Design
-  Dark / Light Theme Toggle
-  Dashboard Analytics Cards
-  Recent Orders Table
-  Sales Analytics Section
-  Recent Updates Panel
-  Modern UI with CSS Variables

-  Sidebar Navigation Menu
-  Mobile Menu Animation

##  Dark Theme

The dashboard includes a custom dark mode implementation using CSS variables.

```css
.dark-theme-variables {
    --clr-color-background: #181a1e;
    --clr-white: #202528;
    --clr-dark: #edeffd;
}
```

Theme switching is handled dynamically with JavaScript.
